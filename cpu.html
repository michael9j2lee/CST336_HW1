The Central Processing Unit (CPU)

The CPU is the control center that is responsible for carrying out tasks

It is also responsible for telling all the other components what to do.

The CPU is generally really small, and is considered to be the heart of the computer

The CPU will often times have a CPU coolder attached with a heatsink.

Each CPU is comprised of a finite number of cores

Each core will be a unit that can process many computing tasks

Some CPU's have integrated graphics that will handle most of the capabilities of GPUs

The speed is measure in GHz
